# Scoring Criteria — Partner Scorecard

Each category is scored 0-100. The weighted total determines the letter grade.

---

## Quality (Weight: 35%)

### Defect Rate Score

| Defect Rate | Score |
|------------|-------|
| 0% (zero defects) | 100 |
| < 0.5% | 90-99 |
| 0.5% - 1.0% | 80-89 |
| 1.0% - 2.0% | 70-79 |
| 2.0% - 5.0% | 50-69 |
| > 5.0% | 0-49 |

### NCR Rate Score

| NCRs per Order | Score |
|---------------|-------|
| 0 NCRs | 100 |
| < 0.1 NCR/order | 90 |
| 0.1 - 0.2 NCR/order | 80 |
| 0.2 - 0.5 NCR/order | 60 |
| > 0.5 NCR/order | 40 |

### First Article Pass Rate Score

| First Pass Rate | Score |
|----------------|-------|
| 100% | 100 |
| 90-99% | 85 |
| 80-89% | 70 |
| 70-79% | 55 |
| < 70% | 40 |

### Customer Returns Score

| Returns Attributable | Score |
|---------------------|-------|
| 0 returns | 100 |
| 1 return | 75 |
| 2 returns | 50 |
| 3+ returns | 25 |

### Quality Category Score Calculation

```
Quality Score = (Defect Rate Score x 0.35) + (NCR Rate Score x 0.25) + (First Pass Score x 0.25) + (Returns Score x 0.15)
```

---

## Delivery (Weight: 30%)

### On-Time Delivery Score

| OTD Rate | Score |
|----------|-------|
| 100% | 100 |
| 95-99% | 90 |
| 90-94% | 80 |
| 85-89% | 70 |
| 80-84% | 60 |
| 70-79% | 45 |
| < 70% | 25 |

### Average Late Days Score (when late)

| Avg Days Late | Score |
|--------------|-------|
| 1-2 days | 85 |
| 3-5 days | 70 |
| 6-10 days | 50 |
| 11-20 days | 30 |
| > 20 days | 10 |
| N/A (never late) | 100 |

### Lead Time Accuracy Score

| Quoted vs. Actual | Score |
|------------------|-------|
| Delivered early or on time | 100 |
| 1-3 days longer than quoted | 80 |
| 4-7 days longer | 60 |
| 8-14 days longer | 40 |
| > 14 days longer | 20 |

### Milestone Update Score

| Updates On Time | Score |
|----------------|-------|
| Always | 100 |
| Usually | 80 |
| Sometimes | 55 |
| Rarely | 30 |

### Delivery Category Score Calculation

```
Delivery Score = (OTD Score x 0.40) + (Late Days Score x 0.20) + (Lead Time Score x 0.25) + (Milestone Score x 0.15)
```

---

## Cost (Weight: 20%)

### Price Competitiveness Score

| Competitiveness | Score |
|----------------|-------|
| Below market (best price) | 100 |
| At market (competitive) | 80 |
| Slightly above market (5-10%) | 60 |
| Above market (10-20%) | 40 |
| Well above market (> 20%) | 20 |

### Price Stability Score

| Price Changes in Period | Score |
|------------------------|-------|
| No increases | 100 |
| 1 increase, < 5% | 80 |
| 1 increase, 5-10% | 60 |
| Multiple increases | 40 |
| Frequent/large increases | 20 |

### Hidden Cost Score

| Unexpected Charges | Score |
|-------------------|-------|
| None | 100 |
| Minor (< 5% of order value) | 75 |
| Moderate (5-10%) | 50 |
| Significant (10-20%) | 30 |
| Major (> 20%) | 10 |

### Cost Category Score Calculation

```
Cost Score = (Price Competitiveness x 0.40) + (Price Stability x 0.30) + (Hidden Cost Score x 0.30)
```

---

## Communication (Weight: 15%)

### Response Time Score

| Average Response | Score |
|-----------------|-------|
| < 4 hours | 100 |
| 4-8 hours | 85 |
| 8-24 hours | 70 |
| 1-2 business days | 50 |
| > 2 business days | 25 |

### Proactive Updates Score

| Proactive Updates | Score |
|------------------|-------|
| Always | 100 |
| Sometimes | 65 |
| Rarely | 35 |
| Never | 10 |

### Language Barrier Score

| Language Issues | Score |
|----------------|-------|
| None | 100 |
| Minor (occasional clarification) | 75 |
| Significant (frequent misunderstandings) | 40 |

### Issue Resolution Score

| Resolution Speed | Score |
|-----------------|-------|
| Fast (same day or next day) | 100 |
| Average (2-3 days) | 70 |
| Slow (> 3 days) | 40 |

### Communication Category Score Calculation

```
Communication Score = (Response Time x 0.30) + (Proactive Updates x 0.30) + (Language x 0.15) + (Issue Resolution x 0.25)
```

---

## Final Score and Grade

```
Final Score = (Quality x 0.35) + (Delivery x 0.30) + (Cost x 0.20) + (Communication x 0.15)
```

| Score | Grade |
|-------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| < 60 | F |
