**DISCLAIMER:** HTS classification requires analysis of each specific product. These are starting points, not guaranteed classifications. Consult a licensed customs broker for binding determinations.

## Important Notes

1. **Parts vs Articles:** A machined part that is clearly a "part of" a specific machine (Chapter 84/85) often gets a lower rate than the same item classified generically by material (Chapters 72-76).

2. **Section 301 impact:** Add 7.5%-25% to ALL rates above for China-origin goods. Check which list the HTS falls under.

3. **AD/CVD watch list:**
   - Aluminum extrusions from China (check scope)
   - Cast iron/steel parts (check scope)
   - Steel wire/rod products (check scope)

4. **Rates change.** Verify against current HTS at hts.usitc.gov.
