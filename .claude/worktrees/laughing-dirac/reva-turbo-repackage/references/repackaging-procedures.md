# Repackaging Procedures

## Standard Repackaging Process

### 1. Receiving and Staging

1. Confirm inspection is complete and passed (check order-state.jsonl for Stage 08 PASS)
2. Stage product in the repackaging area
3. Verify quantity matches inspection report
4. Review the repackaging work order for customer-specific requirements

### 2. Remove Original Packaging

1. Carefully remove China supplier packaging
2. Inspect product for any transit damage not caught during formal inspection
3. Set aside original packaging for disposal or recycling
4. If transit damage found, stop and report to PM — may require NCR

### 3. Clean and Prepare

1. Remove any supplier labels, stickers, or markings not required by customer
2. Clean product as needed (dust, oil residue, fingerprints)
3. Apply protective coatings if specified (rust preventative, protective film)
4. For precision parts, handle with gloves and clean tools

### 4. Inner Packaging

Select based on product requirements:

| Product Type | Inner Packaging | Materials |
|-------------|----------------|-----------|
| Standard metal parts | Bubble wrap | 3/16" or 1/2" bubble wrap |
| Precision machined | Foam insert / custom cut | PE foam, 1-2" thickness |
| Electronics / PCB | Anti-static bag + bubble | ESD bags, pink anti-static bubble |
| Corrosion-sensitive | VCI wrap + desiccant | VCI poly bags, silica gel packs |
| Fragile / glass | Molded foam + double box | Custom foam, inner/outer carton |
| Heavy parts (>50 lbs) | Foam + crate | Wood crate, foam blocking |
| Small fasteners / hardware | Poly bags + carton | Clear poly bags, labeled |

### 5. Outer Packaging

| Method | When to Use | Specifications |
|--------|------------|---------------|
| Standard carton | Default for most shipments | 275# ECT corrugated, size to fit |
| Heavy-duty carton | Items 25-70 lbs | 375# ECT or double-wall corrugated |
| Wood crate | Items >70 lbs or high value | Heat-treated lumber (ISPM 15 compliant) |
| Pallet + stretch wrap | LTL/FTL shipments | Standard 48x40 pallet, 80-gauge stretch wrap |
| Customer-supplied packaging | When specified in PO | Per customer specifications |

### 6. Void Fill

Fill all voids in the package to prevent movement during transit:

- Packing peanuts (biodegradable preferred)
- Kraft paper
- Air pillows
- Foam-in-place (for high-value items)

**Rule of thumb:** Product should not move more than 1/4" in any direction when package is shaken.

### 7. Seal and Label

1. Seal cartons with 2" packaging tape, H-tape pattern (center + both edges)
2. Apply all required labels (see labeling-requirements.md)
3. Apply fragile/orientation stickers if applicable
4. Record package weight and dimensions

### 8. Quality Check

Before marking work order complete:

- [ ] Correct product in correct package
- [ ] Quantity matches work order
- [ ] All labels applied and legible
- [ ] Package properly sealed
- [ ] No visible damage to packaging
- [ ] Weight and dimensions recorded
- [ ] Work order signed off

## Cycle Time Standards

| Order Size | Target Repackaging Time |
|-----------|------------------------|
| 1-10 pieces, simple | 2-4 hours |
| 11-50 pieces, simple | 4-8 hours (1 day) |
| 51-200 pieces, simple | 1-2 days |
| Custom packaging required | Add 1 day |
| Crating required | Add 1-2 days |
| Multiple SKUs / kitting | Add 0.5-1 day |
