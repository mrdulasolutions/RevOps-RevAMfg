# Onboarding User Template

Variables for new customer onboarding.

## Customer Information

- **{{CUSTOMER_COMPANY}}** — Full legal company name
- **{{CUSTOMER_DBA}}** — DBA or trade name (if different from legal name)
- **{{CUSTOMER_WEBSITE}}** — Company website URL
- **{{CUSTOMER_ADDRESS_HQ}}** — Headquarters physical address
- **{{CUSTOMER_ADDRESS_SHIP}}** — Shipping address (if different)
- **{{CUSTOMER_PHONE}}** — Main company phone number
- **{{CUSTOMER_INDUSTRY}}** — Industry / sector
- **{{CUSTOMER_SIZE}}** — Approximate number of employees
- **{{CUSTOMER_DUNS}}** — DUNS number (if available)
- **{{CUSTOMER_ID}}** — Generated customer ID (CUST-YYYYMMDD-NNN)

## Primary Contact

- **{{CONTACT_NAME}}** — Primary contact full name
- **{{CONTACT_TITLE}}** — Job title
- **{{CONTACT_EMAIL}}** — Email address
- **{{CONTACT_PHONE}}** — Direct phone number

## Source and Relationship

- **{{HOW_FOUND}}** — How the customer found Rev A (referral, website, trade show, cold outreach, BD rep)
- **{{BD_REP}}** — BD rep who sourced or manages the relationship (Matt Nebo, Barry Coyle, Bryce Martel, Ryan Knight)
- **{{REFERRAL_SOURCE}}** — Referral source (if applicable — customer name or other source)
- **{{ANNUAL_SPEND_POTENTIAL}}** — Estimated annual spend potential
- **{{ORIGINATING_RFQ}}** — RFQ ID that triggered onboarding (if applicable)

## Legitimacy Verification

- **{{WEBSITE_CHECK}}** — Website verification result (Verified / Not Found / Suspicious)
- **{{EMAIL_CHECK}}** — Email domain match result (Match / Mismatch / Personal Email)
- **{{BUSINESS_CHECK}}** — Business presence confirmation (Confirmed / Unconfirmed / Concerns)
- **{{RED_FLAG_CHECK}}** — Red flag screening result (Clear / Flags Identified)
- **{{LEGITIMACY_DECISION}}** — Overall legitimacy decision (Pass / Caution / Fail)
- **{{LEGITIMACY_NOTES}}** — Notes on any concerns or findings

## PM Assignment

- **{{ASSIGNED_PM}}** — Assigned PM (Ray Yeh or Harley Scott)
- **{{PM_ASSIGNMENT_REASON}}** — Reason for PM assignment
- **{{RAY_ACTIVE}}** — Ray Yeh active order count
- **{{HARLEY_ACTIVE}}** — Harley Scott active order count

## Credit Terms

- **{{CREDIT_TERMS}}** — Established credit terms
- **{{CREDIT_APP_STATUS}}** — Credit application status (Not Requested / Requested / Received / Approved)
- **{{FIRST_ORDER_VALUE}}** — Estimated first order value
- **{{CREDIT_NOTES}}** — Additional credit notes

## Onboarding Metadata

- **{{ONBOARDING_DATE}}** — Date onboarding was completed
- **{{ONBOARDING_STATUS}}** — Status (Complete / Pending / Escalated)
- **{{ESCALATION_FLAG}}** — Whether escalation was triggered
