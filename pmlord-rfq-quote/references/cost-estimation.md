# Cost Estimation Framework

## Estimation Approach

When the PM does not have a China partner quote, use the following estimation framework. These are guideline ranges based on typical Rev A manufacturing work. Actual partner quotes should replace estimates whenever available.

## Estimation Confidence

| Data Source | Confidence |
|-----------|-----------|
| China partner quote | High (use as-is) |
| Previous similar order | Medium-High (adjust for current conditions) |
| Estimation framework (this document) | Medium (present as estimate, note uncertainty) |
| PM gut feel | Low (use only as sanity check) |

Always note the estimation confidence level in the internal cost buildup. If confidence is Medium or lower, recommend the PM get a partner quote before finalizing.
