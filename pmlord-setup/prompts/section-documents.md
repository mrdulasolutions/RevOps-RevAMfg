### Q8: Document Naming Convention

> PMLORD uses this default naming convention for reports:
>
>   PMLORD-{Type}-{YYYY-MM-DD}-{ShortName}.{ext}
>
> Example: PMLORD-Quote-2026-03-30-AcmeBracket.docx
>
> Available tokens:
>   {Type}      — Report type (Quote, Inspection, NCR, Invoice, etc.)
>   {YYYY}      — Four-digit year
>   {MM}        — Two-digit month
>   {DD}        — Two-digit day
>   {ShortName} — Project or customer short name
>   {OrderID}   — Order tracking ID
>   {Author}    — Report author name
>   {ext}       — File extension
>
> A) Use default convention
> B) Custom — enter your pattern
